# Employee Leave Management System (ELMS) - Frontend

This is the React frontend for the Employee Leave Management System, built with Vite and TypeScript.

## Technology Stack
- **Library**: React 19
- **Build Tool**: Vite 7
- **Language**: TypeScript
- **Styling**: Modern CSS / Glassmorphism
- **Charts**: Recharts
- **API Client**: Axios

## Prerequisites
- Node.js (Latest LTS recommended)
- npm or yarn

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

The application will be available at `http://localhost:5173`.

## Features
- **Dashboard**: Personal leave balances and request history.
- **Leave Application**: Form to submit new leave requests.
- **Admin Panel**:
  - **Manage Leaves**: Approve or reject employee requests.
  - **Leave Types**: Configure types and max day limits.
  - **Employee Directory**: View staff list and roles.
  - **Analytics**: Real-time charts for organization-wide leave data.
